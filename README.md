## ūüďĆ About the Project

> *"Banking for Every Community ‚ÄĒ Open, Inclusive, and Digital."*

The Mifos Save Mobile Application is an open-source digital banking app that helps financial institutions manage group-based financial services. This project adds support for Group Banking and Village Savings and Loan Associations (VSLA), enabling field officers to manage Centers, Groups, members, savings, loans, attendance, meetings, and collection activities. It also introduces JLG, GLIM, GSIM, and VSLA features to streamline community banking operations and improve financial inclusion.

The mission: To strengthen financial inclusion by digitizing community banking and group financial services.

## ūüõ†ÔłŹ Tech Stack
The Mifos Save Mobile Application is built using modern cross-platform technologies that prioritize scalability, maintainability, performance, and developer productivity. The architecture follows industry best practices to deliver a robust and consistent experience across platforms.

| **Category**              | **Technology / Framework**              | **Purpose**                                                    |
|---------------------------|-----------------------------------------|----------------------------------------------------------------|
| **Programming Language**  | Kotlin (Multiplatform)                  | Share business logic across Android, iOS, Desktop, and Web while reducing code duplication.             |
| **UI Framework**          | Compose Multiplatform, Jetpack Compose  | Build modern, declarative, and responsive user interfaces across multiple platforms.                               |
| **Architecture Pattern**  | MVI (Model-View-Intent)                 | Ensure predictable state management, unidirectional data flow, and improved maintainability. |
| **Networking**            | Ktor + Ktorfit                          | Perform type-safe API communication with backend services and simplify network integration.                         |
| **Asynchronous Handling** | Kotlin Coroutines + Flows               | Enable asynchronous operations, reactive state updates, and efficient background processing.                    |
| **Dependency Injection**  | Koin                                    | Manage dependencies in a lightweight, modular, and testable manner.          |
| **Database**              | Room and Store5                         | Local database storage, data caching, synchronization, and offline support                 |
| **Serialization**         | Kotlinx Serialization                   | Handle JSON parsing, data transformation, and API model serialization.                                 |
| **Image Loading**         | Coil                                    | Provide fast, memory-efficient image loading and caching.                            |
| **Design System**         | Material 3 + Custom Mifos Design System | Deliver a consistent, accessible, and modern user experience across the application.                                |
| **CI/CD**                 | GitHub Actions                          | Automate builds, testing, code quality checks, and deployment workflows.                      |
| **Project Management**    | Jira + GitHub Projects                  | Support sprint planning, issue tracking, project monitoring, and team collaboration.             |
