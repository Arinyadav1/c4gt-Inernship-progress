C4GT Internship Progress Report · 2026

📌 About the Project ↑ Top

"Banking for Every Community — Open, Inclusive, and Digital."

The Mifos Save Mobile Application is an open-source digital banking app that helps financial institutions manage group-based financial services. This project adds support for Group Banking and Village Savings and Loan Associations (VSLA), enabling field officers to manage Centers, Groups, members, savings, loans, attendance, meetings, and collection activities. It also introduces JLG, GLIM, GSIM, and VSLA features to streamline community banking operations and improve financial inclusion.

The Mission: To strengthen financial inclusion by digitizing community banking and group financial services.

🛠️ Tech Stack ↑ Top

The Mifos Save Mobile Application is built using modern cross-platform technologies that prioritize scalability, maintainability, performance, and developer productivity.

Category Technology Purpose
Language Kotlin (Multiplatform) Share business logic across Android, iOS, Desktop, and Web.
UI Framework Compose Multiplatform Build modern, declarative, and responsive user interfaces.
Architecture MVI (Model-View-Intent) Ensure predictable state management and unidirectional data flow.
Networking Ktor + Ktorfit Perform type-safe API communication with backend services.
Asynchronous Handling Coroutines + Flows Enable asynchronous operations and reactive state updates.
Database Room and Store5 Local database storage, caching, and offline support.
Dependency Injection Koin Manage dependencies in a lightweight, testable manner.
CI/CD GitHub Actions Automate builds, track issues, and support sprint planning.
Design System Material 3 + Custom Mifos Design System Deliver a consistent, accessible, and modern user experience across the application.
Image Loading Coil Provide fast, memory-efficient image loading and caching.
Serialization Kotlinx Serialization Handle JSON parsing, data transformation, and API model serialization.
Project Management Jira + GitHub Projects Support sprint planning, issue tracking, project monitoring, and team collaboration.
